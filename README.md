# imt-fitness-recommendation

Proyek ini adalah aplikasi sederhana berbasis Python untuk menghitung **Indeks Massa Tubuh (IMT)** dan memberikan rekomendasi:
- Kategori berat badan (Kurus, Normal, Berat Badan Berlebih, Obesitas)
- Aktivitas fisik yang sesuai
- Rekomendasi makanan sehat

##Struktur Proyek
- beratbadan.py → Kelas untuk menghitung IMT & rekomendasi makanan
- aktivitas_fisik.py → Kelas turunan untuk rekomendasi aktivitas fisik
- nutrisi.py → (opsional) rekomendasi nutrisi tambahan
- userprofil.py → data pengguna
- main.py → entry point aplikasi
- test.py → file untuk testing
